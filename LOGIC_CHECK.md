# 项目逻辑检查清单

## ✅ 已完成的功能

### 1. 单式模式
- [x] 计算交集：选择多个分类时，显示同时满足所有条件的号码
- [x] 计算并集：显示所有选中号码的合集（去重）
- [x] 交集检查：当选择的条件没有交集时，不允许选中该按钮并提示用户
- [x] 显示格式：
  ```
  统计交集结果：XX, XX, XX，（共X个）
  统计合集结果：XX, XX, XX，（共X个）
  选择条件
  XXX：XX, XX, XX，（共X个）
  ```

### 2. 复式模式（复式二~七）
- [x] 使用并集：将所有选中的号码进行组合
- [x] 去重：自动去除重复号码的组合（如49,49）
- [x] 去重：自动去除相同组合（02,03,04 和 04,03,02 算同一组）
- [x] 组合排序：所有组合按号码升序排列
- [x] 限制显示：最多显示前100注，超过显示省略提示
- [x] 不检查交集：复式模式下不限制用户选择

### 3. 拖式模式（拖式二~七）
- [x] 弹窗选择：用户点击按钮时弹出对话框选择"拖胆"或"拖码"
- [x] 灵活分配：用户可以自由决定哪些号码是拖胆，哪些是拖码
- [x] 组合生成：拖胆中的每个号码 + 拖码中的n-1个号码组合
- [x] 去重：自动过滤拖胆和拖码有重复号码的组合
- [x] 显示格式：
  ```
  统计结果（拖式X）：
  拖胆：XX, XX，（共X个）
  拖码：XX, XX，（共X个）
  共X注
  ```

### 4. 号码分类
- [x] 号码：01-49 直接选择
- [x] 生肖：蛇、龙、兔、虎、牛、鼠、猪、狗、鸡、猴、羊、马
- [x] 五行：金、木、水、火、土
- [x] 波色：红波、蓝波、绿波
- [x] 头数：0头-4头
- [x] 尾数：0尾-9尾
- [x] 两面：大、小、单、双、野兽、家畜、尾大、尾小

### 5. 数据准确性
- [x] 生肖对应号码正确
- [x] 五行对应号码正确
- [x] 波色对应号码正确
- [x] 野兽/家畜分类正确
- [x] 尾大/尾小定义正确

### 6. 用户交互
- [x] 按钮点击：选中/取消选中
- [x] 自动计算：每次选择后自动更新结果
- [x] 清空功能：清除所有选择和结果
- [x] 复制功能：
  - 单式：复制交集号码
  - 复式/拖式：复制所有注数
- [x] 模式切换：切换模式时自动重新计算

### 7. 界面布局
- [x] 计算模式选择器在顶部左侧
- [x] 清空、复制按钮在顶部右侧
- [x] 结果显示框在操作栏下方
- [x] 分类按钮按顺序排列：生肖、五行、波色、头数、尾数、两面、号码
- [x] 拖式对话框：美观的弹窗，两个按钮（拖胆/拖码）
